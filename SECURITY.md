# Security Policy

SSRFHunter Pro is intended only for authorized security testing, penetration testing with permission, and educational use. Users must have explicit permission before scanning systems.

## Supported Versions

This repository is the current development version.

## Reporting a Vulnerability

Report security issues privately by opening an issue and labeling it as `security`, or contacting the maintainers directly.

## Safe Usage

- Do not scan targets without explicit authorization.
- Use rate limiting and timeouts in sensitive environments.
- Do not use SSRFHunter Pro for destructive or unethical testing.
